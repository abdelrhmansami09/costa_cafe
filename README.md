# Costa Cafe Sales Analysis Dashboard

This project is an end-to-end business intelligence dashboard built with Microsoft Excel to analyze sales performance for Costa Cafe. The primary goal is to transform raw sales data into actionable insights to support business decision-making.

## 📊 Project Overview
The dashboard provides a comprehensive view of the cafe's operations, including product popularity, sales trends over time, and operational efficiency across different shifts. It is designed to be interactive, allowing users to filter data by month, day, and payment method using Excel slicers.

## 🚀 Key Features
- **Interactive Filtering:** Users can dynamically explore data using slicers for months, days, shifts, and payment methods.
- **Dynamic Titles:** The dashboard title updates automatically based on the selected month.
- **Visual Storytelling:** Clean and professional charts to highlight key performance indicators (KPIs).

## 💡 Key Insights
- **Top Product:** "Americano With Milk" is the most popular item, driving the majority of sales.
- **Operational Peak:** The "Morning" shift accounts for the highest number of orders, suggesting a need for optimal staffing during this period.
- **Payment Trends:** 93.2% of transactions are cash-based, highlighting a strong reliance on physical currency.
- **Weekly Patterns:** Tuesdays show a notable peak in customer activity.

## 🛠 Tools & Skills
- **Microsoft Excel:** Pivot Tables, Slicers, Data Modeling, and Advanced Formulas (Dynamic Titles).
- **Data Visualization:** Storytelling, chart selection, and UI/UX design principles for dashboards.
- **Data Analysis:** Cleaning, processing, and deriving business insights.
